CSS Flex: moving of html element with respective to the main-axis is called css Flex
--> css flex is one-dimensional
--> by default css flex is responsive.

To Work with flex parent and child relationship is mandatory

CSS FLEX PROPERTIES:
1. display:flex
it is used to convert block level elements to the inline elements

2. gap
it is used to provide gap between html elements
ex: gap:20px;

3. justify-content:
it is used to move the html elements horizontally.
--> default value is flex start

4. align-items: 
it is used to move the html elements vertically.
--> default value is stretch.

5. flex-direction:
it is used to change the direction of the html elements.
--> default value is row.

6. order
it is used to change the order of the html elements.
ex: oder:2

7. flex-wrap:
it is used to wrap the html elements.

8. flex-basis
it used to specify width to the html elements
ex: flex-basis:30%;

9. flex-grow
it is used to specify grow/expand proportion to a particular item 
ex: flex-grow:2;

10. flex-shrink
it is used to specify collapse/compression proportion to a particular item
ex: flex-shrink:2
